Professional integration between Moodle and WooCommerce to synchronize courses, sell them, and enroll users with SSO access.

== Description ==

PCC WooOTEC Chile connects Moodle with WooCommerce to:

* Sync categories and courses from Moodle.
* Create and update WooCommerce products automatically.
* Enroll users when an order is completed.
* Generate SSO access per course.
* Show a frontend "My Courses" dashboard with templates.
* Keep logs in the uploads directory.

== Installation ==

1. Upload the plugin ZIP in Plugins > Add New.
2. Activate WooCommerce before activating this plugin.
3. Configure Moodle URL, token and default options.
4. Run an initial manual sync.
